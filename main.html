What's live vs. sample

Live: the impact analysis and comment drafting are real — each runs an LLM reasoning pass against the profile above when you click.

Sample: the docket feed is seeded with real, representative 2026 rulemakings. In production it polls the regulations.gov v4 API for open comment periods on a schedule (the ingestion code mirrors the Congress.gov pattern: filter dockets by commentEndDate, queue matches, re-check daily).

Deploying it for real

Drop this file on any static host. Two production wires:

1 · a regulations.gov API key for the live feed.

2 · route the LLM calls through a backend that holds the Anthropic key, rather than the in-app demo path used here.
